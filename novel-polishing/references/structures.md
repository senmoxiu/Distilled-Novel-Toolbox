# 应避免的句式结构

本文件列出所有应当消除的AI腔句式结构。

## 双重对比句式

通过人为制造虚假的戏剧性冲突来引出观点。请直接、正面地陈述核心观点。

| 句式模式 | 问题 |
|----------|------|
| "不是因为X。而是因为Y。" | 套路化的刻意转折铺垫 |
| "X不是问题。Y才是。" | 公式化的观点重新包装 |
| "答案不是X。而是Y。" | 极其死板的观点过渡 |
| "这看起来像X。其实是Y。" | 故弄玄虚的"揭晓"陈词滥调 |
| "问题不在于X。而在于Y。" | 修辞上的刻意误导 |
| "不是X。而是Y。" | 机械化的生硬对比 |
| "不是这个。而是那个。" | 换汤不换药的同样公式 |
| "不再是X，开始成为Y" | 刻意制造的虚假转型弧光 |
| "并不意味着X，而实际上是Y" | 先否定再肯定的啰嗦垫话 |
| "是关于X，而不是Y" | 虚假/无意义的二分法划分 |
| "不仅是X，而且也是Y" | 增加字数的对冲修饰 |
| "不仅仅关乎X，它其实是Y" | 同上，AI高频滥用 |

**替代方案：** 直接陈述Y。"问题在于Y。""这里Y很关键。"完全省去前半句的否定铺垫。

## 否定列举

在透露事物"是什么"之前，先罗列一堆它"不是什么"。

| 句式模式 | 问题 |
|----------|------|
| "不是X……不是Y……而是Z。" | 通过层层否定制造戏剧性铺垫 |
| "这不是X。这不是Y。这是Z。" | 同上句式 |

**替代方案：** 直接陈述Z。读者不需要这么长的起跑线铺垫。

## 戏剧化断句

故意将完整句子切碎以示强调，充满人工雕琢的"伪深刻"。

| 句式模式 | 问题 |
|----------|------|
| "[名词]。就是这样。这就是该事物。" | 表演式虚假极简主义 |
| "X。以及Y。以及Z。" | 刻意顿挫感的短句堆叠 |
| "这释放了某种东西。[某个词]。" | 刻意制造的"顿悟"效果 |

**替代方案：** 使用完整的句子。信任内容本身的价值，不靠排版和标点故弄玄虚。

## 设问句与修辞铺垫

在预示"我要说出真知灼见了"，而不是直接讲出来。

| 句式模式 | 问题 |
|----------|------|
| "如果[重新定义]会怎么样？" | 苏格拉底式故作姿态，显得傲慢 |
| "我的意思是：" / "以下是我的意思：" | 冗余的预告说明 |
| "想想看：" / "细思极恐：" | 居高临下的说教式催促 |
| "这没关系。" / "这也完全可以接受。" | 毫无必要的单方面"恩准"与心理安慰 |

**替代方案：** 直接抛出观点，让读者自己得出结论。

## 公式化陈词滥调

| 句式模式 | 问题 |
|----------|------|
| "到了X的时候，我已经是Y了。" | 俗套的小说式叙事模板 |
| "不是Y的X"（如"无法带来改变的举措"） | 绕弯子。直接说"X失败了" |

## 虚假施动者（无生命主语）

给无生命事物赋予人类动作。AI特别偏爱这种表达，因为可以含糊其辞地避开具体动作执行人。

| 句式模式 | 问题 |
|----------|------|
| "意见变成了解决方案" | 意见自己什么也做不了，是有人解决了它 |
| "一个尝试在几天内决定生死" | 尝试没有生命，是人做出了决策 |
| "决定逐渐浮现" | 决定不会自己浮现，是人做出了决定 |
| "文化发生了转变" | 文化不会自己转变，是人的具体行为改变了 |
| "对话转向了……" | 对话不会自动移动，是谈话的人引导了方向 |
| "数据告诉我们" | 数据冷冰冰地躺着，是人读取并得出了结论 |
| "市场会奖赏……" | 市场没有情感，是买家付费购买了产品 |

**替代方案：** 明确指出是哪个人做的。用"团队在当周解决了这个问题"代替"意见变成了解决方案"。

## 旁观者视角

像上帝一样漂浮在场景上方做抽象总结（讲师腔/上帝腔）。

| 句式模式 | 问题 |
|----------|------|
| "没有人设计这个。" | 脱离现实的凭空观察 |
| "这之所以发生，是因为……" | 居高临下的讲师般说教腔调 |
| "这就是为什么……" | 同上 |
| "人们往往会……" | 键盘社会学家的抽象分析 |

**替代方案：** 将读者置于具体场景中。用"你不会某天突然坐下来，然后决定去……"代替"没有人设计这个"。

## 被动语态

每个句子都需要一个动作执行者。被动语态往往隐藏了动作的真实执行人。

| 被动句式 | 建议改用 |
|----------|----------|
| "X被创建" | 明确指出是谁创建了X |
| "人们普遍认为……" | 明确指出是谁这么认为 |
| "犯了错误" | 明确指出是谁犯的错 |
| "得出了决定" | 明确指出是谁做的决定 |
| "无需配置文件。结果会被自动保存。" | "你不需要准备配置文件。系统会自动保存结果。" |

**替代方案：** 找出动作的主动执行者，放在句首。

## 应避免的句首词

| 句首模式 | 修改建议 |
|----------|----------|
| **疑问词开头**：以"如何""什么""为什么""哪里""当……时"等引导的倒装或冗余句式 | 重新调整句式，从主语或动词直接开始。如"让这件事变得困难的是其高昂的成本"→"高昂的成本阻碍了这一举措" |
| **以"所以"开头的段落** | 直接从实质性的内容开始陈述 |
| **以"看""听着"开头的句子** | 直接删掉这些虚假语气词 |

## 肤浅套路句式

AI喜欢在句尾强行追加一串分句来伪造深刻分析。

**高警惕词汇：** 凸显了、确保了、折射出、体现了、有助于、培养了、展示了

**修改前：**
> 该寺庙采用蓝色、绿色和金色的配色方案，与该地区的自然美景相呼应，象征着德克萨斯州的美人草、墨西哥湾和多样的德克萨斯景观，折射出社区与土地的深厚纽带。

**修改后：**
> 该寺庙使用了蓝、绿、金三色。建筑师表示，这些颜色的灵感来自当地的美人草和墨西哥湾海岸。

## 模板式"挑战与前景"段落

AI写作极度喜欢在文末塞公式化的"面临挑战但前景光明"总结。

**高警惕词汇：** 尽管面临……但依然、面临着多重挑战、挑战与传承、未来展望

**修改前：**
> 尽管工业繁荣，科拉图尔也面临着城市地区常见的挑战，包括交通拥堵和水源短缺。尽管面临这些挑战，凭借其战略位置和正在实施的举措，科拉图尔继续作为金奈增长不可或缺的一部分而蓬勃发展。

**修改后：**
> 2015年三个新IT园区开放后，当地交通拥堵加剧。市政公司于2022年启动了雨水排放工程，以解决频发的洪涝问题。

## 避用简单谓语（绕弯子）

AI不喜欢直接说"是"或"有"，喜欢用复杂的长词绕弯子。

**高警惕词汇：** 充当了……的角色、作为……而存在、标志着、呈现出……的特点、提供了、具备

**修改前：**
> 825画廊作为洛杉矶艺术协会的当代艺术展览空间而存在。该画廊呈现出四个独立空间的特点，具备超过3000平方英尺的面积。

**修改后：**
> 825画廊是洛杉矶艺术协会的当代艺术展厅。该画廊有四个房间，总面积为3000平方英尺。

## 虚假关联（从X到Y）

AI喜欢用"从X到Y"句式，但X和Y不在同一逻辑维度。

**修改前：**
> 我们穿过宇宙的旅程，带领我们从大爆炸的奇点走向宏大的宇宙网，从恒星的诞生与死亡走向暗物质的神秘舞蹈。

**修改后：**
> 本书涵盖了大爆炸、恒星演化以及当前关于暗物质的理论。

## 同义词翻来覆去（词义重复）

AI为了避免词汇重复，强行用不同词指代同一事物。

**修改前：**
> 主角面临着诸多挑战。这位主要人物必须克服重重阻碍。这位中心人物最终获得了胜利。英雄回到了家乡。

**修改后：**
> 主角面临重重挑战，但最终获得胜利回到了家乡。

## 节奏模式

| 套路节奏 | 修改建议 |
|----------|----------|
| **三项式排比/列举** | 极具AI腔调。改用两项或一项 |
| **自问自答**（提出问题并立即回答） | 要么删掉问题改为陈述句，要么让问题留有余地 |
| **每个段落都以金句收尾** | 节奏千篇一律，请丰富段落结尾的长度和语气 |
| **滥用破折号（—）** | 彻底移除。改用逗号或句号 |
| **零碎断句的堆砌** | 不要连续堆叠三四个极短句 |
| "并非总是如此。也并不完美。" | 伪装成真诚安抚的对冲套话，直接删掉或给出具体界限 |

## 词汇模式

| 禁用词汇模式 | 问题 |
|--------------|------|
| **偷懒的极端绝对词**（每一个、总是、从不、所有人、没有人） | 空泛的虚假权威感。请用具体事实或范围代替 |
| **所有修饰副词**（真的、只是、简直、真正地、诚实地、仅仅、实际上） | 无意义的苍白强调，稀释了句子的力量 |
