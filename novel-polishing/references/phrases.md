# 禁用词组清单

本文列出所有应当消除的AI腔词组。

## 垫话开场白

删除这些"做预告式"的过渡句，直接陈述内容本身。

- "其实：" / "事情是这样的："
- "以下是……""这就是……""那就是……"
- "这就是为什么……"
- "残酷的事实是" / "令人不安的事实是"
- "事实证明" / "原来"
- "真正的……是"
- "坦白说" / "让我说清楚"
- "事实上，"
- "我再说一遍："
- "老实说" / "说实话"
- "我们能聊聊……吗"
- "我觉得有趣的是"
- "但问题在于"
- "毫无疑问，……"
- "众所周知，……"
- "不可否认的是，……"
- "值得一提的是，……"
- "正如我们所知，……"

所有包含"这里是/以下是/也就是说"的句式结构，都是切入正题前的废话。请直接删掉它们。

## 交互式聊天废话

- "希望这能帮到你！"
- "当然！" / "没问题！"
- "你说的太对了！"
- "你想不想……"
- "随时告诉我"
- "问得好！"
- "如果你想让我展开任何章节，请随时告诉我。"

## 刻意的强调语气

- "句号。" / "就是这样，没得商量。"
- "好好想想吧。" / "细品一下。"
- "这很重要，因为……"
- "别搞错了"
- "这就是为什么这很重要"
- "这不仅是……更是……"
- "我们必须认识到……"

## 表演式强调

- "悄然爬入" / "悄悄渗透"
- "我保证"
- "它们确实存在，我向你保证"

## 商业行话 / 大厂黑话

| 避免使用 | 建议改用 |
|----------|----------|
| 导航/跨越（挑战） | 处理、应对 |
| 拆解（分析） | 解释、剖析 |
| 倾斜进入（拥抱） | 接受、拥抱 |
| 格局/全景（背景） | 现状、领域 |
| 游戏规则改变者/颠覆性 | 重大的、重要的 |
| 双倍下注/加倍投入 | 致力于、加大力度 |
| 深潜/深挖/深度剖析 | 深入分析、探究 |
| 退后一步/抽离出来 | 重新考虑 |
| 向前迈进/展望未来 | 接下来、从现在起 |
| 绕回来/闭环 | 重新讨论、回到 |
| 在同一页上/达成共识 | 达成共识、一致 |
| **赋能** | 帮助、支持 |
| **闭环** | 完成、做好、形成循环 |
| **打法** | 方法、策略 |
| **对齐** | 沟通一致、确认同步 |
| **抓手** | 切入点、突破口、核心手段 |
| **心智**（如占领心智） | 用户习惯、品牌认知 |
| **底层逻辑** | 基本原理、核心逻辑 |
| **赛道** | 行业、领域 |
| **痛点** | 问题、困难、急需解决的需求 |

## AI高频词汇

- 实际上、此外、至关重要、探索/深入探索、强调、持久的、提升、培养
- 收获/赢得、凸显、相互作用、错综复杂/精妙之处、关键的
- 格局/蓝图（抽象用法）、核心/纽带、编织/织锦（抽象比喻，如"历史的织锦"）
- 见证/证明、彰显、宝贵的、充满活力的

## 过度渲染重要性词汇

- 充分体现了、生动诠释了、不可或缺的角色、里程碑、深远影响
- 奠定了基础、标志着、勾勒出、关键转折点、不断变化的格局
- 不可磨灭的印记、根深蒂固

## 宣传性与广告式词汇

- 拥有、充满活力、丰富的（比喻）、深厚的、提升了其、展示了、彰显了
- 致力于、自然美景、坐落于、位于……的心脏地带、开创性的、著名的
- 令人窒息的、必去之地、极具吸引力的

## 模糊指代词汇

- 行业报告显示、观察家指出、专家认为、一些批评家指出
- 几家媒体/文献（但实际只引用了一两个）
- 独立报道、地方/区域/国家媒体广泛报道、由知名专家撰写
- 活跃的社交媒体互动

## 知识库截止免责与猜测填充

- 截至[日期]、在我最近一次训练数据更新中
- 虽然具体细节有限、根据现有信息
- 未公开、保持低调、对个人生活保密、避开聚光灯
- 大概[成长于/学习了/开始于]、人们相信

## 副词与修饰词

删掉所有的修饰性副词，尤其是中文"地"字修饰词：

- 真的 / 真正地、仅仅 / 只是、字面意义上 / 简直
- 真诚地 / 真正地、老实说 / 诚实地、简单地 / 仅仅
- 实际上 / 其实、深深地 / 深度地、真正地 / 真实地
- 根本上 / 从根本上说、固有地 / 本质上、不可避免地
- 有趣的是、重要的是、关键的是

同时删除以下填充性词组：

- "其核心是" / "本质上"
- "在当今的……中" / "如今"
- "值得注意的是" / "需要说明的是"
- "归根结底" / "说到底"
- "当谈到……时" / "关于"
- "在一个……的世界里" / "当今"
- "现实是" / "实际上"

## 元评论

- "提示："
- "剧情反转：" / "剧透："
- "你可能已经知道了，但……"
- "但这可以另写一篇文章了"
- "……是个特性，而不是漏洞"
- "包装成……"
- "本文的其余部分将解释……"
- "让我带你了解……"
- "在这一节中，我们将……"
- "正如我们将看到的……"
- "我想探讨……"
- "让我们深入了解""让我们探讨""让我们拆解"
- "这里是需要知道的"
- "现在让我们看"
- "废话少说"

## 说教而非展示

- "这真的很困难"
- "这就是领导力真正的样子"
- "这就是……真正的样子"
- "其实很重要"

## 含糊的陈述

- "其原因在于结构性问题"
- "其影响是深远的"
- "这是最核心/最深层的问题"
- "利害攸关" / "代价高昂"
- "后果是真实的"

**法则：** 如果一句话只是空洞地告诉你某件事"很重要""影响深远"或"是结构性的"，却拿不出具体的例子或事实，请直接删掉，或用具体细节代替。

## 谄媚与迎合

- "问得好！"
- "你说的非常正确"
- "这是一个很复杂的话题"
- "你的观点太棒了"

## 冗余填充性词组

| 避免使用 | 建议改用 |
|----------|----------|
| 为了实现这一目标 | 为此 |
| 由于当时下雨了的事实 | 因为下雨 |
| 在当前的这个时间点上 | 现在 |
| 在您需要帮助的情况下 | 如需帮助 |
| 该系统具备处理……的能力 | 该系统可以处理…… |
| 需要指出的是，数据显示…… | 数据显示…… |
