# 润色指令模板库

## 完整去AI化流程指令

使用以下流程指令让AI执行完整的去AI腔改写：

### 标准流程指令

```
请对以下文本进行去AI腔润色，按以下流程执行：

1. 识别AI模式：扫描文本，找出39条规则中每一条的具体实例
2. 撰写初步改写稿
3. 使用评分系统评分（五个维度各1-10分）：
   - 直截了当：直接陈述还是预告铺垫？
   - 节奏变化：句子长短有致还是千篇一律？
   - 信任读者：是否过度解释或说教？
   - 真实感：有具体细节和真实情感吗？
   - 信息密度：有可删减的虚词或黑话吗？
4. 列出残留的AI特征
5. 输出最终改写稿

硬性约束：最终稿中不能出现任何破折号（—）。
```

### 声音微调指令

如用户提供写作样本：

```
请先分析以下写作样本的句子长度模式、词汇水平、段落开头习惯、标点偏好、过渡句处理方式，然后在改写中模仿此风格：[样本内容]
```

未提供样本则使用默认风格：长短句交错、直截了当、富于变化、自然口语化。

## 网文分场景润色

### 通用去AI化

```
请对以下网文段落进行去AI腔润色：
1. 减少过于完美的句式结构
2. 添加适当不规则表达
3. 使用个性化语言
4. 偶尔用口语化表达
5. 避免机械化段落结构
6. 保留核心信息和意图
7. 使用2-3成复合句
8. 掺入5%口语填充词
9. 禁用破折号
10. 禁用大厂黑话（赋能/闭环/对齐等）
```

### 战斗场景

```
请润色以下战斗场景：
1. 增强动作描写的画面感
2. 使用短句增强节奏
3. 添加感官细节（触觉/听觉/嗅觉）
4. 减少过度解释
5. 加入主角即时感受和判断
6. 让动作本身说话
7. 避免AI式"剑气凌厉""杀意滔天"等套话
```

### 情感场景

```
请润色以下情感场景：
1. 增加内心独白的真实感
2. 使用不完整思绪和犹豫
3. 添加身体反应描写（心跳/手抖/呼吸）
4. 减少直白情感陈述
5. 用行为和细节暗示情感
6. 增加矛盾和挣扎
```

### 对话场景

```
请润色以下对话：
1. 增加口语化程度
2. 添加语气词和停顿
3. 为每个角色设计独特的说话方式
4. 增加潜台词和言外之意
5. 减少信息量过大的对话
6. 让对话更自然流畅
```

### 环境描写

```
请润色以下环境描写：
1. 增加感官层次（五感）
2. 用具体细节替代抽象描述
3. 让环境暗示情绪
4. 减少静态描写，增加动态元素
5. 用比喻增强画面感
6. 避免AI式"如画""令人窒息"等宣传腔
```

### 人物描写

```
请润色以下人物描写：
1. 用具体特征替代抽象评价
2. 通过动作/习惯展现性格
3. 避免AI式"英俊潇洒""气质非凡"模板
4. 用细节制造画面感和记忆点
5. 让外貌描写服务于人物塑造
```

## 评分评估指令

```
请对以下改写稿进行五维评分（每项1-10分）：
1. 直截了当：是否直接陈述事实，无预告铺垫？
2. 节奏变化：句子长短有致吗？排除三项式排比了吗？
3. 信任读者：尊重读者智商吗？避免过度解释/猜测填充？
4. 真实感：像真人写的吗？有具体细节和情绪吗？
5. 信息密度：有可删减的虚词/黑话/空泛展望吗？

列出评分并说明扣分原因。总分低于35分需重新改写。
```
